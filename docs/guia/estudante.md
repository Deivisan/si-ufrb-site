### 2. E-mail Institucional

- **Formato**: `seu_nome@estudante.ufrb.edu.br`
- **Acesso**: https://webmail.ufrb.edu.br
- ** problema comum**: Erro de autenticação
  - Solução: Redefinir senha pelo portal do calouro

### 3. Plataformas de Aulas

#### RNP Mconf (Recomendado)
```
✅ Vantagens:
- Integração com AVA
- Gravações automáticas
- Sala fixa por disciplina

📋 Passo a passo:
1. Acesse o AVA
2. Entre na disciplina
3. Clique em "Aulas Síncronas"
4. Clique no link do Mconf
5. Aguarde o início da aula
```

#### Google Meet (Backup)
```
✅ Usado quando Mconf falha
🔗 Links alternativos estão em /atividades
```

## 📤 Entrega de Atividades

### Formatos Aceitos
- PDF (documentos)
- ZIP (códigos)
- Link (repositórios GitHub)

### Prazos
- **Default**: Sábado às 23:59
- **Prorrogado**: Caso haja problema no AVA

### Dica
```bash
# Antes de entregar:
1. Verifique o prazo
2. Faça backup no GitHub
3. Envie pelo AVA
4. Guarde o comprovante
```

## 📈 Acompanhamento de Desempenho

### Método de Estudo Recomendado (Pomodoro Adaptado)

```
25 min estudo + 5 min descanso
Depois de 4 ciclos: 15-30 min descanso
```

### Ferramentas

- **Anki**: Flashcards (Lógica, Matemática)
- **Notion**: Organização de estudos
- **GitHub**: Versionamento de código
- **Obsidian**: Conexões de conhecimento

## 🚨 Problemas Comuns e Soluções

| Problema | Solução |
|----------|---------|
| AVA fora do ar | Use prazos prorrogados |
| E-mail não funciona | Redefinir senha pelo portal |
| Aula não carrega | Use Meet alternativo |
| Quiz com erro | Reporte ao professor |
| Gravação não aparece | Espere 24h após aula |
