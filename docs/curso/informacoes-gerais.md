### 📅 Duração e Estrutura

- **Turno**: EAD (Ensino a Distância)
- **Polos**: Feira de Santana, Santo Amaro, Ipirá

### 📚 Projeto Pedagógico (PPC)

O currículo está alinhado com os seguintes eixos:

1. **Fundamentos** - Lógica, Algoritmos, Matemática
2. **Desenvolvimento de Sistemas** - Programação, Banco de Dados
3. **Gestão de Projetos** - Metodologias Ágeis, Soft Skills
4. **Pesquisa e Inovação** - TCC, Iniciação Científica

### 🎯 Objectivos de Aprendizagem

Ao concluir o curso, o estudante será capaz de:

- Desenvolver sistemas computacionais de médio e grande porte
- Aplicar metodologias ágeis de desenvolvimento de software
- Trabalhar em equipe e comunicar-se efetivamente
- Analisar e resolver problemas técnicos e de negócio
- Projetar e implementar soluções inovadoras

### 📖 Disciplinas por Período

#### 1º Semestre
- Algoritmos e Programação I
- Fundamentos de Sistemas de Informação
- Lógica Matemática
- Matemática Discreta

#### 2º Semestre
- Algoritmos e Programação II
- Arquitetura de Computadores
- Base de Dados I
- Introdução às Ciências de Dados

*(Continuar até o 8º semestre)*
